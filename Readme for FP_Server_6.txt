- FP_Server_6 is for raw image capture and enhancement.
- Low power mode is disabled here and expectectation is to turn off the device manually after use.
- Raw images can be collected rapidly, for this turn on the device switch and use the provided charger plugged in to AC socket.

-For enabling the configuration Access point "FPS_Config" make sure the server system is in "Airplane mode".

-Long beep with 3 times blue ring light flashing denotes device is powered on.
-Purple ring light flashing indicates server is not configured properly or any other connectivity issues.
-Yellow ring light flash once and white ring light glow steadily indicating the device is ready for inget print input.
